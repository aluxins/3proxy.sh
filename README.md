# 3proxy.sh - bash script to install and setup 3proxy

## Supported OS:
* Centos 7

## Functional:
* Install 3proxy.
* Remove 3proxy
* Add the specified number of users
* Delete one or all users
* Re-issue all users

## 1. How to install

```
curl -O https://raw.githubusercontent.com/intens1ty/3proxy.sh/master/3proxy.sh
chmod +x 3proxy.sh
```
